# Diabetes Analyzer

## Overview
The Diabetes Analyzer is a comprehensive application designed to help users track, manage, and analyze their diabetes-related data. It provides insights into blood sugar levels, medication schedules, and lifestyle choices to promote better health and well-being.

## Features
- **Data Tracking**: Log daily blood sugar levels, medication intake, and meal details.
- **Reports and Analysis**: Generate reports to visualize trends in blood sugar levels over time.
- **Medication Reminders**: Set reminders to take medications and manage schedules.
- **Nutrition Guides**: Access dietary recommendations and meal suggestions based on individual needs.

## Setup
### Prerequisites
- Python 3.x installed on your machine
- A virtual environment (recommended)

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/markvader/DiabetesAnalyzer.git
   cd DiabetesAnalyzer
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the application:
   ```bash
   python app.py
   ```
2. Navigate to `http://localhost:5000` in your web browser to access the interface.
